📚 Formules Utilisées

Loi Normale (Z)

P(X < x) = Φ((x-μ)/σ)
z = Φ⁻¹(p)

Loi de Student (t)

P(T < t) = CDF(t, ddl)
tα = CDF⁻¹(α, ddl)

Loi du Chi-2 (χ²)

P(χ² > x) = 1 - CDF(x, ddl)

Loi de Fisher (F)

P(F > x) = 1 - CDF(x, ddl1, ddl2)

Tests d'Hypothèse

Si p-value ≤ α → Rejet de H₀
Si p-value > α → Non rejet de H₀

Test de Shapiro-Wilk

W = (Σaᵢ × x_(i))² / Σ(xᵢ - x̄)²

Test de Jarque-Bera

JB = n/6 × (g₁² + g₂²/4)
g₁ = Skewness, g₂ = Excès de Kurtosis
